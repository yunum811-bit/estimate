# ระบบประเมินพนักงาน (Employee Evaluation System)

## วิธีใช้งานในองค์กร

### ขั้นตอนที่ 1: เปิด Server (ทำครั้งเดียวบนเครื่องหลัก)

```bash
cd c:\Web\ประเมิน
npm start
```

จะแสดง IP Address ให้ เช่น:
```
เปิดในเครื่องนี้:
http://localhost:3000

ให้คนอื่นในองค์กรเข้าผ่าน:
http://192.168.1.100:3000
```

### ขั้นตอนที่ 2: แจก URL ให้คนในองค์กร

ส่ง URL เช่น `http://192.168.1.100:3000` ให้ทุกคนเปิดในเบราว์เซอร์
(ทุกคนต้องอยู่ WiFi/LAN เดียวกัน)

## หมายเหตุ
- ข้อมูลเก็บในไฟล์ `db.json` (เปิดดู/แก้ไขได้)
- ถ้าอยากรีเซ็ตข้อมูล ให้ลบ `db.json` แล้วรัน server ใหม่
- Server ต้องเปิดค้างไว้ตลอดที่ใช้งาน
